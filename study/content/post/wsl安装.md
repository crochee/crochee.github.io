### windows版本要求
安装wsl2对系统版本有硬性的要求
* 运行 Windows 10（已更新到版本 2004 的内部版本 19041 或更高版本）
* 通过按 Windows 徽标键 + R，检查你的 Windows 版本，然后键入 winver，选择“确定”。 如果内部版本低于 19041，请更新到最新的 Windows 版本
